## Запуск программы

### Maven

Для сборки проекта:

```sh
mvn clean compile assembly:single
```

Для запуска проекта:

```sh
java -jar .\target\week-5-v1-jar-with-dependencies.jar
```

## Задание
Для своего проекта применить на практике паттерны Singleton, Factory/Factory method, Strategy
