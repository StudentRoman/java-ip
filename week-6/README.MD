## Запуск программы

### Maven

Для сборки проекта:

```sh
mvn clean compile assembly:single
```

Для запуска проекта:

```sh
java -jar .\target\week-6-v1-jar-with-dependencies.jar
```

## Задание
Для своего проекта применить на практике паттерны Builder, Adapter и Iterator
